---
title: "Establishments"
description: "Manage establishment (store) records: list, create, retrieve by ID, and update client time-in-store targets."
---

The establishments API manages the physical store locations visited by field operators. Functions in `Establishment.js` use several namespaces: `EstablecimientosNuevos/` for establishment CRUD, `clientes/` for client lookups, `usuarios/` for user lookups, `formularios/` for form lookups, `regiones/` for region lookups, and `analytics/` for time-in-store configuration.

<Note>
  All requests require a valid Bearer token. See [Authentication](/api/authentication) for setup instructions.
</Note>

---

## getClients

Fetch all clients. Used in the establishment management UI to populate client filter dropdowns.

**Method:** `GET`  
**Path:** `/clientes/getClients`

### Request

No parameters required.

### Response

Returns the full Axios `response` object. Access records via `response.data`.

### Example

<CodeGroup>

```javascript getClients
import { getClients } from "@/Api/Establishment";

const response = await getClients();
console.log(response.data);
```

</CodeGroup>

### Request

<ParamField body="dataFinal" type="object" required>
  The complete establishment object to persist. The exact schema is defined by your establishment model.
</ParamField>

### Example

<CodeGroup>


## updateTimeInStore

Update the target time-in-store for one or more client-establishment combinations. This targets the analytics extras endpoint.

**Method:** `POST`  
**Path:** `/analytics/extras/updateTimeInStore`

### Request

<ParamField body="data" type="object" required>
  An object describing the time-in-store update. The exact shape depends on your analytics schema — typically an array of `{ client, establishment, targetMinutes }` entries.
</ParamField>

### Response

Returns the full Axios `response` object.

### Example

<CodeGroup>

```javascript updateTimeInStore
import { updateTimeInStore } from "@/Api/Establishment";

const response = await updateTimeInStore([
  { client: "client-xyz", establishment: "store-001", targetMinutes: 30 },
]);
```

</CodeGroup>
