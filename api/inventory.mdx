---
title: "Inventory"
description: "Manage inventory records: list, save, transfer, and track movements. Includes lookups for clients, users, forms, regions, and promoter users."
---

The inventory API covers all inventory management operations. Functions in `inventory.js` use the `inventario/`, `clientes/`, `usuarios/`, `formularios/`, and `regiones/` namespaces.

<Note>
  All requests require a valid Bearer token. See [Authentication](/api/authentication) for setup instructions.
</Note>

---

## getClients

Fetch all clients. Used to populate client filter dropdowns in the inventory UI.

**Method:** `GET`  
**Path:** `/clientes/getClients`

### Request

No parameters required.

### Response

Returns the full Axios `response` object. Access records via `response.data`.

### Example

<CodeGroup>

```javascript getClients
import { getClients } from "@/Api/inventory";

const response = await getClients();
console.log(response.data);
```

</CodeGroup>

## transferInventario

Transfer inventory between users or locations.

**Method:** `POST`  
**Path:** `/inventario/transferInventario`

### Request

<ParamField body="data" type="object" required>
  A transfer object describing the source, destination, items, and quantities to move. The exact schema is defined by your inventory model.
</ParamField>

### Response

Returns the full Axios `response` object.

### Example

<CodeGroup>

```javascript transferInventario
import { transferInventario } from "@/Api/inventory";

const response = await transferInventario({
  from: "user-uid-123",
  to: "user-uid-456",
  items: [{ sku: "PROD-001", qty: 10 }],
});
```

</CodeGroup>

## saveInventario

Persist an inventory snapshot or update.

**Method:** `POST`  
**Path:** `/inventario/saveInventario`

### Request

<ParamField body="data" type="object" required>
  An inventory object to save. The exact schema depends on your inventory model.
</ParamField>

### Response

Returns the full Axios `response` object.

### Example

<CodeGroup>

```javascript saveInventario
import { saveInventario } from "@/Api/inventory";

const response = await saveInventario({
  user: "user-uid-123",
  items: [{ sku: "PROD-001", qty: 50 }],
  date: "2026-03-20",
});
```

</CodeGroup>
